Class: PingOneConsents

PingOneConsents()

Class representing consent management and enforcement. This is done with a custom PingOne schema using JSON attributes and PingAuthorize. This demo-specifc class is developed and maintained by PingIdentity Technical Enablement. Implements methods to integrate with PingOne and PingAuthorize-related API endpoints.
